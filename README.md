# 🖥️ Infinix OS

A modern **browser-based operating system** that replicates a real desktop experience inside the browser.

---

## 🚀 Overview

Infinix OS is an interactive web-based OS built using modern frontend technologies.
It provides a desktop-like environment where users can open apps, manage windows, and interact with the system just like a real operating system.

---

## ✨ Features

* 🔐 Lock Screen, Login & Register
* 🪟 Window Management (minimize, resize, fullscreen)
* 📁 File Manager
* 📝 Notes App
* 🌐 Built-in Browser *(limited by web security policies)*
* 📅 Calendar & Time
* 🔋 Real-time Battery Status
* 📶 WiFi Indicator UI
* 🎨 Customizable UI (themes & wallpapers)
* 🖱️ Draggable & Interactive Desktop
* 🪟 Multiple apps running simultaneously (multi-window support)

---

## 🛠️ Tech Stack

* React
* Tailwind CSS
* Framer Motion
* Browser APIs (Battery, Date/Time)
* Python

---

## 🎯 How It Works

The system simulates an OS using a component-based architecture where each app behaves like a window.
User interactions such as dragging, resizing, and opening apps are handled dynamically using state management and event-driven logic.
